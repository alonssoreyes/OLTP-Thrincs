## Install and exec

Ensure you have installed Python and Pip

### Create a virtual environment (optional but recommended)

Windows:

```bash
python -m venv env
env\Scripts\activate
```

macOS/Linux:

```bash
python3 -m venv env
source env/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Execute

Windows:
```bash
python main.py
```

macOS/Linux:
```bash
python3 main.py
```

## Tests

```bash
pytest
```
